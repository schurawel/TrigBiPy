# TrigBiPy Paper: Incoherence is all you need

Dieses Verzeichnis enthält das fortlaufende LaTeX-Manuskript für das Paper **"TrigBiPy: Incoherence is all you need."**.

## Worum geht es?
Der Artikel skizziert die Architektur von „TrigBiPy“, einem formalen System zur Validierung menschlicher Erkenntnisprozesse. Entgegen traditionellen Ansätzen postuliert TrigBiPy, dass Erkenntnisgewinn als deterministisches System epistemischer Zustandsübergänge (Morphismen) beschrieben werden kann. Durch die Reduktion auf vier fundamentale Kategorien (Quelle, Beobachtung, Theorie, Diskurs) wird eine geschlossene algebraische Struktur definiert. Das System fungiert als *epistemischer Linter*, der logische Inkohärenzen aufdeckt.

Dies ist übrigens eng mit dem Projekt "tree(3)"(https://github.com/schurawel/tree3) verbunden, wobei jenes ein Ansatz zur praktischen Anwendung ist. Jedoch ist die Theorie im Paper unter anderem aus Erkenntnissen die das tree(3) Projekt bis jetzt gebracht hat entstanden, und ist diesem um Lichtjahre vorraus.
